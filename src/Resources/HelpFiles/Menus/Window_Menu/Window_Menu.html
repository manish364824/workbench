Window Menu


The Window Menu contains options for viewing Viewing Tabs and Workbench Windows.

The Window Menu contains:

  • Next Tab switches to the Viewing Tab to the right of the currently displayed tab.  This can also be done with the key strokes Command + right arrow.

  • Previous Tab switches to the Viewing Tab to the left of the currently displayed tab.  This can also be done with the key strokes Command + left arrow.

  • Rename Selected Tab... allows the user to rename the Viewing Tab with any name or descriptor they choose.

               

  • Move All Tabs in Current Window to New Windows separates all of the Viewing Tabs out into individual Workbench Windows.

  • Move All Tabs From All Windows Into Selected Window takes all individual Workbench Windows, turns them into Viewing Tabs and joins them into a single Window.

  • Information Window opens the Information box.  When the user clicks on a surface or volume in the viewing area, this is where the information for that location is displayed.

  • Scenes... opens the Scenes  box.  When there is a check mark in front of this option (as shown below), it means the Scene box is open.  Click this option again and it will close the Scenes box.

  • Bring All To Front pulls all of the Workbench entities (Windows, ToolBoxes, etc.) that are opened, out in front of all other displays (Terminal windows, browsers, applications, etc.) making them unhidden.

  • Tile Windows organizes all open Workbench Windows into columns and rows.
