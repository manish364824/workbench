Help Menu


The Help Menu contains  access to information that is meant to assist the user with any questions they have about Workbench.

The Help Menu contains:

  • Search allows the user to type in a keyword and get direct access to that specific information.  For example, type in "tool" and a list of choices that include the word tool appear.

         

  • Workbench Help... you're looking at it.  The list on the left are links to general information about Workbench's makeup, such as the Menus.

          

  • Go to HCP Website... will open humanconnectome.org in a new browser window.

  • Report a Workbench Bug... is a convenient form the user can fill out and submit if a bug occurs.

         

  • Submit HCP Software Feature Request... opens http://humanconnectome.org/contact/feature-request.php for an online feature request form.  All suggestions on how to make the HCP Software more robust are welcome.
