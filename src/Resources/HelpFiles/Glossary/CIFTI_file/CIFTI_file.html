CIFTI file


CIFTI is a new file format that stores data from surfaces (vertices) and volumes (voxels) concurrently in a single file comprising a listed set of brainordinates.  The volume component of a CIFTI file can be any selected list of voxels (e.g., only subcortical gray-matter voxels) that need not conform to cuboidal or ‘N x M x O’ volumetric dimensions.  See http://www.nitrc.org/projects/cifti/ for a detailed exposition of the CIFTI file format.  CIFTI files are consistent with the NIFTI-2 file format and by convention have the extension .nii.  Workbench currently supports CIFTI files for dense scalar (*.dscalar.nii), dense time series (*.dtseries.nii), and dense connectomes (*.dconn.nii).
