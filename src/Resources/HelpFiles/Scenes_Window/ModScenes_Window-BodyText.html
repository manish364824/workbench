Scenes Window

The Scenes Window allows the user to save the current Workbench session (loaded files and settings) in a file as a Scene.  Often views in Workbench require several steps to set up.  Scenes allow the user to save a snapshot of a Workbench session that contains a record of all of the files loaded and view settings for that session making it easy to get to a specific display.  By reopening a saved Scene file, a user can "pick up where they left off" when they reopen Workbench.
  • Scene File drop-down menu is used when there are multiple Scene files loaded into Workbench.
  • New... button allows the user to create and save a new Scene File.  The new Scene File will have no scenes within it, to begin with, so scenes will need to be added.
  • Show will load the active (highlighted in blue) scene.  This can also be done by double clicking on the scene.
  • Preview... opens a separate small window with an enlarged display of the highlighted scene's thumbnail along with the name and description.
  • Add... creates a new scene within the current Scene File and adds it to the bottom of the list of scenes.  The scene needs to be set up before clicking this button.
  • Insert... is similar to Add... because it creates a new scene but this adds the new scene above the currently highlighted scene.
  • Replace... is used when the user wants to modify an existing scene and save the changes.  The changes would be made prior to clicking this button.  The option to change the name is also available at this point. 
  • Delete... allows the user to delete a highlighted scene from the Scene File.
